Spotify Clone

A Spotify clone made in React native using the Spotify Web APIs that lets you log into your spotify account and handle all the playlist related operations .

git repo :-

https://github.com/SiddhantRN/Spotify

Scan to open

With an Android phone, you can scan this QR code with your Expo mobile app to load this project immediately.

Description and Functionalities

This app is used to manage spotify playlists, some of the main functionalities are :-

Get user's playlists After logging in to your spotify account you'll get a list of your playlists.

Create new Playlist

Modify Playlist Rename an existing playlist.(changing the scope of playlist coming in next update)

List Tracks of a playlist

Delete a track Swipe left on any track to delete it from the playlist.

Framework and Libraries used

  • React Native
  • React Native Navigation
  • Expo AuthSession
  • Fetch Api
  • Expo Secure Store
  • React Native Modalize
