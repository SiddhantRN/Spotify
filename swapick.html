Description and Functionalities

Some of the main functionalities of this app are :-

One item at a time Swipable cards( Tinder like UI ) with all the necessary item deatails so that you can focus on one item at a time.

In app chat Chat with the seller or buyer through the app itself. No need to exchange contact information anymore.

Sell Have something you don't use anymore ? Great!! Sell it or swap (exchange) it with something you like.

Some Other features Use filters to browse the type of items you are interested in, save items for later and much more.

Framework and Libraries used

  • React Native
  • React Native Navigation
  • Formik
  • Joi
  • React Native Modalize
  • React Native Reanimated
